스크린샷 저장기

Windows 스크린샷 단축키 (Win + Shift + S)로 캡처한 이미지를
클립보드가 아닌 바탕화면의 폴더에 자동으로 저장해주는 도구입니다.

다운로드

설치 가이드

1
다운로드한 ZIP 파일의 압축을 풀어주세요.
압축을 푼 폴더를 바탕화면에 가져다 놓으세요.
2
바탕화면에 "AutoScreenshots" 폴더를 만들어주세요.
이 폴더에 스크린샷들이 자동으로 저장됩니다.
중요: 위의 과정이 완료될 때까지 압축 해제된 폴더 안의 어떤 파일도 열지 마세요!
3
Python을 설치해주세요.
Python 공식 사이트에서 최신 버전을 다운로드하여 설치하세요.
4
프로그램을 실행해주세요.
압축 해제한 폴더 안의 easy_setup.bat 파일을 더블클릭하여 실행하고 완료될 때까지 기다려주세요.
자동 시작 설정 (선택사항):
컴퓨터를 켤 때마다 자동으로 실행되게 하려면 다음 단계를 따르세요:
5
바로가기 만들기
easy_setup.bat 파일에서 마우스 우클릭 → "바로 가기 만들기"를 선택하세요.
6
시작 폴더로 이동
Win + R을 누르고 shell:startup을 입력한 후 Enter를 눌러주세요.
7
바로가기 복사
5단계에서 만든 바로가기를 열린 시작 폴더로 이동시켜주세요.
완료! 이제 Windows 스크린샷 단축키 (Win + Shift + S)를 사용할 때마다
캡처한 이미지가 바탕화면의 "AutoScreenshots" 폴더에 자동으로 저장됩니다.

문제가 발생하면 프로그램을 다시 실행해보세요.
